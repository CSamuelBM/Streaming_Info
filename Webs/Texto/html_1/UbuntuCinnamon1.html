Ubuntu Cinnamon

Es un "sabor" oficial de Ubuntu que combina la estabilidad y robustez de Ubuntu con el entorno de escritorio Cinnamon (popularizado por Linux Mint).

Ofrece una interfaz moderna pero tradicional y familiar, ideal para usuarios que migran desde Windows, con un uso de recursos moderado.

Es una opción excelente si prefieres la base técnica de Ubuntu pero no te agrada el entorno GNOME predeterminado.
